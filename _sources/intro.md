# Hello Universe

This repository contains a draft Jupyter notebook for the Hello Universe project, an initiative at MAST/STScI to provide tutorials applying machine learning methods to astronomical data.

The notebook in this preview is:

**Exploring Astronomy Word Associations with Small Language Models**

This tutorial uses an astronomy-themed Codenames task to introduce semantic embeddings, cosine similarity, prompt sensitivity, and small-language-model behavior. It is designed as a lightweight educational notebook that follows the Hello Universe tutorial style: learning goals, introduction, imports, numbered tutorial sections, exercises, FAQ/extensions, About this Notebook, and citations.
